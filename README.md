## 💡 简介

该仓库用于展示一些有趣有用的新项目，旨在：

* 帮助大家发现新鲜有趣有用的开源项目
* 帮助开发者推广开源项目 

欢迎开发者来投稿，也欢迎大家来推荐自己觉得不错的开源项目！

## ✨ 什么是“新项目”

1. 项目是在 GitHub 上开源
2. 项目星数区间为 [10, 500]

## 🎨 如何投稿

可通过[创建 issue](https://github.com/88250/awesome-seeds/issues/new) 或者发起 Pull Request 来进行投稿。
